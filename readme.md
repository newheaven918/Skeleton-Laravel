# Laravel  5.1 Skeleton 

To use this project, make a fork

## Admin user and password

```
user: admin@domain.com
pass: admin
```

## How to install

First make storage folder writable. 

```
cp .env.example .env
```

```
composer install
```

```
php artisan key:generate
```

```
php artisan migrate --seed
```

# Front-end development

```
npm install
```

```
gulp watch
```

## Make production

```
gulp --production
```
